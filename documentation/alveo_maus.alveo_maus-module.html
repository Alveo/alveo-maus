load_lexicon(lexdirpath=None)

source code 

Load all files in the specified directory as a lexicon dictionary. Each line of each file is split by whitespace, then the last delimited element is treated as the phonetic representation, while the rest are joined with spaces, converted to lowercase, then recorded as the orthographic representation

Parameters:
  • lexdirpath (String) - the directory to search for lexicon files. If not specified, this will default to the value in the config file.
Returns: Dict
the combined lexicon, as a dictionary

Parameters:

  • lex (Dict) - the lexicon to use; if unspecified, the lexicon specified in the configuration file will be loaded and used
Returns: String

Raises:
  • IncompleteLexiconError - if there is a word appearing in the orthographic transcript that is not covered by the lexicon
